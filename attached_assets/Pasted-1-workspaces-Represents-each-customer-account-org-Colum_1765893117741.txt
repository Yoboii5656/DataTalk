1. workspaces
Represents each customer account/org.
Column | Type | Description
id | UUID (PK) | Workspace ID
name | TEXT | User/company name
owner_id | UUID | Reference to users.id
created_at | TIMESTAMP |
plan | TEXT | free/pro/enterprise
status | TEXT | active/suspended

2. users
Column | Type
id | UUID (PK)
email | TEXT
name | TEXT
workspace_id | UUID (FK -> workspaces.id )
role | TEXT
created_at | TIMESTAMP

3. agents
Represents each voice agent configured.  
Column | Type
id | UUID (PK)
workspace_id | UUID (FK)
name | TEXT
language | TEXT (en, hi, gu, etc.)
llm_model | TEXT
status | TEXT
created_at | TIMESTAMP

4. integrations
All integrations (CRM, telephony, vector DB, webhook) stored here.  
Column | Type
id | UUID (PK)
workspace_id | UUID (FK)
type | TEXT (hubspot, zoho, twilio, plivo, slack, custom_api)
config | JSONB
status | TEXT (active, inactive, error)
last_sync_at | TIMESTAMP
created_at | TIMESTAMP

5. agent_tools
Tools attached to an agent (CRM, Search, Email Sender, HubSpot API, custom scripts).  
Column | Type
id | UUID
agent_id | UUID (FK -> agents.id)
tool_name | TEXT
tool_config | JSONB
created_at | TIMESTAMP

6. agent_runs
Each time an agent is executed.  
Column | Type
id | UUID (PK)
agent_id | UUID (FK)
workspace_id | UUID (FK)
run_type | TEXT (live_call, test_call, cron_job, webhook)
status | TEXT (success, failed, partial)
duration_ms | INT
started_at | TIMESTAMP
completed_at | TIMESTAMP

7. test_runs
Used when a user clicks "Run Test" for an agent.  
Column | Type
id | UUID
agent_id | UUID (FK)
workspace_id | UUID (FK)
test_input | TEXT
expected_output | TEXT
actual_output | TEXT
result | TEXT (pass, fail)
error_message | TEXT
created_at | TIMESTAMP

8. run_logs
Stores logs streamed during agent run.  
Column | Type
id | UUID
run_id | UUID (FK -> agent_runs.id)
step | INT
event_type | TEXT (llm_call, tool_call, user_input, system)
message | TEXT
payload | JSONB
timestamp | TIMESTAMP

9. errors
Tracks all system-level or run-level failures.  
Column | Type
id | UUID
run_id | UUID (FK)
workspace_id | UUID (FK)
source | TEXT (integration, agent, llm, tool)
code | TEXT
message | TEXT
metadata | JSONB
created_at | TIMESTAMP

11. billing_usage
Column | Type
id | UUID
workspace_id | UUID
agent_id | UUID
characters_generated | INT
calls_made | INT
tokens_used | INT
total_cost_usd | DECIMAL
created_at | TIMESTAMP

12. audit_events
Tracks config changes, who did what.  
Column | Type
id | UUID
workspace_id | UUID
user_id | UUID
action | TEXT
entity | TEXT (agent, integration, tool)
before | JSONB
after | JSONB
created_at | TIMESTAMP
